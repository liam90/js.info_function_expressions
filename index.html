JS.info Function Expressions

As the function creation happens in the context of the assignment expression (to the right side of =), this is a Function Expression.

let sayHi = function() {
console.log('Hello World');
}

sayHi(); // The output bring Hello World

We can even copy functions to other variables as well

let myFunc = sayHi; myFunc(); // This will give us the same result as sayHi which is "Hello World"
